É necessário ter java instalado. 
Para instalar o Java : sudo apt install openjdk-18-jdk
Após extrair o projeto é possível compilar utilizando : javac Main.java Arvore.java No.java
E executar utilizando e passando o diretorio dos arquivos a serem executados  : java Main <diretorio 1> <diretorio 2> 
Clique com o botao direito no arquivo elementos, selecione copiar caminho e cole no <diretorio 1>. faça o mesmo para o arquivo comandos e coloce no <diretorio 2>.
Deve se parecer com : C:\Users\...\EDB2\Implementacao_ABB\src\elementos.txt ou /home/.../EDB2/Implementacao_ABB/src caso esteja no Linux Assim, o comando deve ficar parecido com :
java Main C:\Users\...\EDB2\Implementacao_ABB\src\elementos.txt C:\Users\...\EDB2\Implementacao_ABB\src\comandos.txt
